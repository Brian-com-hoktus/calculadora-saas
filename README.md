# Calculadora SaaS

Interfaz web para calcular el costo de planes escalables según volumen de conversaciones, revisiones de antecedentes, documentos y contratos.

## Cómo usar

1. Instala dependencias:
   ```bash
   npm install
   ```
2. Inicia la aplicación en modo desarrollo:
   ```bash
   npm run dev
   ```
3. Abre el navegador en `http://localhost:4173`.

## Despliegue en Vercel

Este proyecto es una app Vite lista para desplegar en Vercel.

## Características

- Cálculo automático según volumen de conversaciones.
- Tarifas escaladas por tramos.
- Estimación de personas reclutadas, revisiones, páginas de documentos y contratos.
- Selección de plan con base fee y descuento.
